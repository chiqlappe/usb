# USB入力デバイスをPC-8001で使う

## 概要
- USBジョイスティックやトラックボールなどの一般的な入力デバイスを、PC-8001で使用可能にするプロジェクト
- USBホストインターフェースとしてラズパイpicoを採用
- PC-8001とのインターフェースとして8255(PPI)を採用

## インターフェース基板について
- ```Gerber```内の```USB.zip```を使用して基板を作成して下さい
- 抵抗やロジックICなどの部品は基板に記載されているものを使用して下さい
- トランジスタはECBタイプのNPN型を使用して下さい
- 使用する際には「USB Aメス-マイクロUSB BオスのOTGケーブル」が必要です

## ラズパイpico用プログラムについて
- ```usb_hid_board_rev1```プロジェクトをArduino IDEでコンパイルし、picoに書き込んで下さい
- ツールメニュー内の ```USB Stack``` を ```Adafruit TinyUSB Host(native)``` にして下さい
- ライブラリが不足している場合は、エラーメッセージを参考にしてインストールして下さい
   
## 仕様
### I/Oポート
|ポート番号|説明|データ方向|
|:--:|:--|:--:|
|$8C|データ受信用|入力|
|$8D|未使用||
|$8E|制御信号用|出力|
|$8F|PPI設定用|出力|

### 制御信号
|ビット番号|名称|説明|
|:--:|:--|:--|
|0|CS|モード切り替え|
|1|CLK|クロック|
|2|未使用||
|3|未使用||
|4|未使用||
|5|未使用||
|6|未使用||
|7|未使用||

### PPIのモード
|ポート名|モード|方向|
|:--:|:--|:--:|
|A|MODE0|入力|
|B|未使用||
|C-H|未使用||
|C-L||出力|

## USBから取得できるデータについて
- データは8ビット単位(0~255)で取得できます
- データには入力デバイスが取得した移動量やボタンの押下状態などが記録されています
- データの位置や総数はデバイス毎に異なります

## データ取得方法

### 初期設定
- PPIにコントロールワード$90を出力します
```
OUT &H8F,&H90
```

### 取得手順(N-BASICの場合)
1. 制御信号を以下の通りに送信します
```basic
OUT &H8E,3 'CS=H,CLK=H
OUT &H8E,1 'CS=H,CLK=L
OUT &H8E,3 'CS=H,CLK=H
```
2. ポートAからデータ総数を受信します
```basic
A=INP(&H8C)
```
3. 制御信号を以下の通りに送信します
```basic
OUT &H8E,2 'CS=L,CLK=H
OUT &H8E,0 'CS=L,CLK=L
OUT &H8E,2 'CS=L,CLK=H
```
4. ポートAからデータを受信します
```basic
A=INP(&H8C)
```
5. データ総数分だけ3~4を繰り返します

### 動作確認用サンプルプログラム（N-BASIC）

```basic
10 'initialize PPI
20 OUT&H8F,&H90
30 'send CLOCK signal while CS is HI
40 OUT&H8E,3
50 OUT&H8E,1
60 OUT&H8E,3
70 'receive report number
80 N=INP(&H8C)
90 IF N=0 OR N=&HFF THEN 40
100 FOR I=1 TO N
110 'send CLOCK signal while CS is LOW
120 OUT&H8E,2
130 OUT&H8E,0
140 OUT&H8E,2
150 'receive report
160 D=INP(&H8C):PRINT RIGHT$("0"+HEX$(D),2)+" ";
170 NEXT I
180 PRINT
190 GOTO 40

```

## 動作確認
1. 抵抗、ロジックIC、トランジスタ、PPI(8255)、picoなどの部品が正しく取り付けられていることを確認して下さい
2. picoのマイクロUSB B端子にOTGケーブルを接続します
3. USBデバイスをOTGケーブルに接続します
4. PC-8001mk2の電源を切り、基板を拡張スロットに挿入します。基板の底が金属部分に触れないよう注意して下さい
5. 本体の電源を入れて正しく起動することを確認します。異常を感じたら直ちに電源を切り、基板を確認して下さい
6. 動作確認用のサンプルプログラムを入力して実行します。USBデバイスからの入力が検出される度に表示が更新されます
7. USBデバイスからのデータが取得できていることを確認して下さい。デバイスの中にはこの基板でデータ取得できないものもありますので、複数のデバイスでチェックすることをお勧めします
